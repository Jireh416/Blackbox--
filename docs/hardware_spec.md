# 15³ 情緒黑盒：硬體規格與配線手冊 (Hardware Specification & Wiring Guide)

這份文件詳細記錄了情緒黑盒內部所有元件的腳位定義與供電架構。**嚴禁未經確認即修改此配線架構。**

## ⚡ 供電架構 (Power Architecture)

本系統嚴格區分控制邏輯電源與動力電源，由「麵包板專用電源模組」作為心臟：
- **電源輸入**：6.5V ~ 12V DC 變壓器輸入至電源模組。
- **5V 系統軌道**：將模組跳線帽設定在 **5V**，供應給麵包板兩側的紅/藍軌道。
- **12V 升壓軌道**：(選配) 若使用 12V 元件，將 5V 經過 MT3608 升壓至 12V 專供。

---

## 🔌 核心控制器腳位定義 (Pinout)

| Arduino UNO Q Pin | 連接模組 / 腳位 | 備註功能 |
| :--- | :--- | :--- |
| **GND** | 麵包板藍線 (-) | 建立系統共地 (Common Ground) |
| **A4 (SDA)** | MPR121 & PCA9685 | I2C 資料線 (並聯) |
| **A5 (SCL)** | MPR121 & PCA9685 | I2C 時脈線 (並聯) |
| **D2 (RX)** | DFPlayer Mini (TX) | 序列埠通訊接收 |
| **D3 (TX)** | DFPlayer Mini (RX) | 序列埠通訊發送 (需串接 1K 電阻) |
| **D4** | 1470C-RGB (WS2812) (DIN) | 驅動狀態燈條 (建議串接 330Ω 電阻保護) |
| **D5 (PWM)** | 扁平震動馬達 | 動態麻刺感震動 (⚠️ **必須透過 MOSFET 驅動**) |
| **D6** | 3cm 微型散熱風扇 | 冷卻期氣流引導 (⚠️ **必須透過 MOSFET 驅動**) |
| **D7** | 5V 超音波霧化片板 (EN) | 控制白霧噴發 |

## 🧰 詳細模組接線指引

### 1. I2C 神經網絡 (MPR121 & PCA9685)
- **供電**：兩者 VCC 皆接 5V 軌道，GND 接 GND 軌道。
- **PCA9685 馬達供電**：綠色接線柱的 `V+` 與 `GND` 必須連接到 5V 軌道與 GND 軌道，提供獨立推力。
- **伺服馬達**：2 顆 SG90 分別插入 PCA9685 的 `Channel 0` 與 `Channel 1`。棕色對齊 GND，紅色對齊 V+，橘色對齊 PWM。

### 2. 聽覺與氣氛模組
- **DFPlayer Mini**：
  - VCC / GND 接至 5V 軌道。
  - 喇叭連接至 `SPK_1` 與 `SPK_2`。
  - SD 卡內需預載 5 首 mp3 檔案對應 5 個階段 (Phase 0 ~ Phase 4)。
- **超音波霧化片**：
  - 驅動板電源接至 5V 軌道。控制腳位 (EN/SIG) 連接至 `D7`。

### 3. 高負載動力元件 (警告)
> [!CAUTION]
> **震動馬達 (D5)** 與 **風扇 (D6)** 絕對不可以將其正負極直接插入 Arduino 的數位腳位！
> 即使是 5V 風扇，其啟動電流 (Inrush Current) 仍會瞬間燒毀 Arduino 的 IO 接腳。
> **解法**：必須使用 N-Channel MOSFET 模組或 NPN 電晶體（如 TIP120）作為電子開關，將控制訊號接到 D5/D6，由 MOSFET 切換 5V 軌道或 12V 升壓軌道的動力電。
